 expr :  KEYWORD:VAR IDENTIFIER EQ expr
                : term(plus | minus)term *

 term : factor(mul | div )factor *

factor :     (plus | minus)factor 
                   : power

power : atom(power factor )*

atom : int float
       lPAREN expr RPAREN

program : statements

statements : - (NEWLINE+ expr)*


SHOW Statement

Syntax:
SHOW expression

Description:
Evaluates an expression and displays its value to the user.

Examples:

SHOW 5 + 2

Output:
7

HENSU a = 10
HENSU b = 30

SHOW a + b

Output:
40


Input expression
Syntax :
 

HENSU var = Input

Examples:
HENSU age = INPUT
SHOW age


WHILE loop

syntax :
WHILE  condition DO  
       statements
END

Examples :

HENSU x = 1

WHILE x <= 3 DO;
SHOW x
HENSU x = x + 1
END

Output:
1
2
3

FOR LOOP

Syntax:

FOR identifier = start_value TO end_value DO
    statements
END

Example:

FOR i = 1 TO 5 DO
    SHOW i
END
